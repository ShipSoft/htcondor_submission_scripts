Production of HNL Samples in FairShip using EventCalc. 

### Prerequisites
Preproduction (.dat files) produced using standalone [EventCalc-SHiP](https://github.com/maksymovchynnikov/EventCalc-SHiP) and converted to .root format using [convertEvtCalc.py macro in FairShip](https://github.com/ShipSoft/FairShip/blob/master/macro/convertEvtCalc.py).
